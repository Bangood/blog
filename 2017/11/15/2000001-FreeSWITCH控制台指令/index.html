FreeSWITCH控制台指令

2017年11月15日 星期三

控制台指令

指令 作用
sofia status profile internal 列出某个Profile的状态
sofia status profile internal reg 列出某个Profile上所有已注册用户
sofia status profile internal 1000 过滤某些符合条件的用户
sofia profile external rescan 添加SIP网关后执行该命令，使新配置生效
sofia status 显示SIP网关注册状态
reloadxml 重新加载配置文件，使新配置生效
global_getvar 查看vars.xml加载前设置的全局变量
sofia global siptrace on/off 开启/关闭SIP消息打印
show codec 查看FreeSWITCH支持的编码类型
sofia loglevel all 9 开启详细的Sofia SIP 底层调试信息

名词解释

名词 描述
IVR Interactive Voice Response，即互动式语音应答
PBX Private Branch Exchange，用户级交换机
PSTN Public Switched Telephone Network，公共交换电话网络
SBC Session Border Controller，会话边界控制器
FQDN Fully Qualified Domain Name，完全合格域名/全称域名
LDAP Lightweight Directory Access Protocol，轻量目录访问协议
